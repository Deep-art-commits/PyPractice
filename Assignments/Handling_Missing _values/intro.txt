Raw data is inconsistent and may have null values 
reasons - data collection errors

Missing completely at random - Missing Completely at random is a type 
of missing data mechanism in which the probability of 
a value being missing is unrelated to both missing data and observed data 

Missing at random - the probability of a value being missing is dependent on the observed data not on missing value 
MAR- eg ,income details on a survey , disability details on application forms 

MNAR= Missing not at random-- probability of missing value depends on 
the missing data itself

3 . Mode imputation - Categorical values
